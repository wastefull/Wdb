### Three Sustainability Dimensions

WasteDB visualizes materials across three dimensions:

- **Recyclability (CR)** - Yellow/green tones
- **Compostability (CC)** - Pink/coral tones
- **Reusability (RU)** - Blue/teal tones

### Visualization Modes

- **Simplified**: Single score bar (0-100)
- **Scientific**: Practical vs Theoretical with confidence intervals
- **3D Halo**: Interactive radar-style visualization
